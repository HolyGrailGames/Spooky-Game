# Terrain-Generator
A Terrain generator made with OpenGl and LibGDX.

## Controls

```
Look around:            Mouse or trackpad/touchpad
Move:                   WASD keys
Fly up:                 R
Fly down:               F
Toggle wireframe mode:  P
Close program:          Esc
```
